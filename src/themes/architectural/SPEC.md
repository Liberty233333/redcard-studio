主题代号: `architectural`
风格: 模拟「现代建筑事务所工作图纸」——描图纸 + 蓝铅笔 + 红铅笔 + 网格底纸
画幅: 3:4 恒定（450×600 预览 / 1080×1440 出图，spec 中 720×960 缩放因子 0.625）

## 设计哲学

不是装饰性"建筑风插画"，是**模拟真实图纸文档**的视觉系统。每个元素都有制图学意义：
- 建筑制图惯例（尺寸标注线、4 角对位、内对角虚线、FIG 编号、元数据条）
- 普鲁士蓝 + 砖红的"红蓝铅笔"传统
- 现代描图纸（vellum）的色调

## 色板（不可改）

```
--paper:    #F0ECE0   现代描图纸暖白
--img-bg:   #E2DDCB   图框待绘区
--prussian: #1E3A5F   主字 / 标注线 / 网格 / 标题 / 副标 / footer 左 / [A]-[D] 标号 / IMG 标注
--brick:    #B23A2C   4 角角标 / FIG 编号 / 「」高亮 / 引线 / N° DRAFT / [V] PORTRAIT
```

砖红只用在"重点标注"——红铅笔位置。

## 字体（不可改）

| 用途 | 字体 | 字号(720) | 字重 | spacing |
|---|---|---|---|---|
| 主标题 | PingFang SC | 60 | 800 | -1 |
| 主标题高亮「」 | PingFang SC | 60 | 200 italic | -1 |
| 副标题 | PingFang SC | 22 | 600 | 0.5 |
| meta / 标号 | mono | 9 | 400 | 2 |
| FIG / [V] | mono | 11 | 700 | 3-4 |
| A NOTE ON TASTE | sans | 11 | 400 | 3 |
| Footer | mono | 10 | 400/700 | 2 |
| IMG 占位 | mono | 11 | 400 | 3 |

主标题 800 + 高亮 200 italic 形成"重锤 + 细线"的字重张力。

## 三种画幅 / variant

- **FIG.01 / 1:1**（方形配图，封面 / 主图）：图区 420×420 居中
- **FIG.02 / 16:9**（横长配图）：图区 680×382 居中
- **FIG.03 / 9:16**（竖长配图）：图区 270×480 居左

## 必须保留的"建筑感真元素"

1. 网格底纹（10px 细 + 40px 粗，普鲁士蓝 0.05/0.10）
2. 配图区内对角虚线（dasharray 4 4）
3. 配图区 IMG 占位标注 `IMG · 宽 × 高`
4. 顶部宽度尺寸标注线（带两端竖小线 + 中间数字位）
5. 侧边高度尺寸标注线（数字旋转 -90°）
6. 4 角砖红角标（crop marks）
7. FIG 编号 `— FIG.0X —`
8. 主标题底部砖红短引线（2pt × 60px）+ A NOTE ON TASTE — 提示词 副标
9. Footer 两端：左 `你的账号 / NOTES`（蓝），右 `N° DRAFT 0X`（红）

## 必须删除的冗余（千万别加回）

- ✗ 顶部 SCHEMA · NOTES / DRAWING N° A-04 大标题栏
- ✗ 顶部/底部双线分隔
- ✗ SECTION A / SECTION B 段标
- ✗ DETAIL 03 / REJECTION 引线注释
- ✗ 副标题前 ⓵ 圆圈编号
- ✗ Footer 上方双线

横线越少越好。除主标题底下那条红短引线（必留），其他横线都是尺寸标注线。

## 标题语法

- 主标题里 `「关键词」` 包裹的内容会渲染成 weight 200 italic 砖红色高亮
- 例：`AI 审美提升\n就差一句「我不要」` → "我不要" 是高亮
