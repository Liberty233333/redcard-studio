## 选 anchor 的逻辑（给代码层参考）

```pseudo
function selectAnchors(paletteFamily, bgMode, count=2):
  candidates = filter(samples where palette == f"{paletteFamily}-{bgMode}")
  if len(candidates) == 0:
    return []  // 该 family + bgMode 未验证，模型靠 prompt 中的 token + 风格描述发挥
  if len(candidates) <= count:
    return candidates  // 全部返回
  return random_shuffle(candidates)[:count]  // 随机选 count 张
```

注意：未验证的 amber + blue 4 个槽位会返回空 anchor 数组，由 prompt 中的 token 描述和构图原则承担风格指引职责。

Anchor 的使用边界：
- Anchor 只提供风格语法，不提供模板。
- 可以学习大字重量、色块比例、主视觉面积、留白密度和杂志/新闻封面的气质。
- 不得复制 anchor 的人物、人脸、文字、固定版式、固定边角标签或固定底部模块。
- 不得复制或生成 carousel 圆点、底部长条、页码点和小红书 UI 元素。
- 同一篇封面要根据 source 素材重新组织构图，避免任何“套版式”的 Canva 感。
